# About This Project

## 📝 Overview
This is a small **Vanilla JavaScript** To‑Do List web app. It lets users add tasks, mark them as completed, and delete them. It is designed to be simple and easy to extend.

👉 [Click here to view the live project](https://garychamp.github.io/to-do-app/)

## 🧩 Project Structure
- `index.html` — The main HTML page that contains the UI.
- `stylesheet.css` — Basic styling for the app (layout, button styles, completed task styles).
- `javascript.js` — App logic (add tasks, mark complete, delete tasks, handle keyboard input).

## 🚀 How to Run
1. Open `index.html` in any modern browser.
2. Type a task into the input field.
3. Click **Add** or press **Enter** to add the task.
4. Click a task to mark it complete (strike-through).
5. Click **Delete** to remove a task.

## ✅ Features
- Add new tasks
- Mark tasks complete (toggle)
- Delete tasks
- Add via button or pressing Enter

## 🔧 Notes / Default Behavior
- Tasks are not persisted across page reloads (no local storage).
- Empty tasks are ignored.

## 📦 Possible Enhancements
- Persist tasks using `localStorage`.
- Add edit task functionality.
- Add task priority or due date.
- Add task filtering (all / active / completed).
- Improve styling/responsiveness.
